Building
========

Applications for the ``m2gl025_miv`` board configuration can be built as usual
(see :ref:`build_an_application`).
In order to build the application for ``m2gl025_ev``, set the ``BOARD`` variable
to ``m2gl025_ev``.

.. code-block:: bash

   export BOARD="m2gl025_ev"
